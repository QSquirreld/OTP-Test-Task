# SQL-1: Эффективность стратегий взыскания (7 дней)

## Задача
Оценить эффективность стратегий взыскания в первые 7 дней после старта кейса.

## Что считалось
По `strategy_name` были рассчитаны:
- `total_cases`
- `contacted_cases_7d`
- `paid_cases_7d`
- `collected_amount_7d`
- `avg_collected_per_paid_case_7d`
- `collection_rate_7d`

## Подход
1. Отобраны кейсы со `start_date` в диапазоне `2026-02-01` — `2026-02-28`
2. Для каждого кейса определены:
   - наличие контакта в первые 7 дней
   - наличие положительного платежа в первые 7 дней
   - сумма положительных платежей в этом окне
3. Для `collection_rate_7d` сумма долга считалась отдельно от платежей, чтобы избежать дублирования `principal_amount` при нескольких платежах на один кейс

## Допущение
Окно 7 дней реализовано как:
- `event_dttm::date BETWEEN start_date AND start_date + 6`
- `payment_dttm::date BETWEEN start_date AND start_date + 6`

То есть от `start_date` до `start_date + 6 дней` включительно.

## Результаты

### Общая картина
- Всего кейсов: 1114
- В выборке 4 стратегии:
  - `late_stage`
  - `mixed`
  - `sms_first`
  - `soft_call`
- Метрики выглядят консистентно: нет аномалий вроде отрицательных значений или rate > 1

### Contact rate (косвенно)
Доля контактированных кейсов:
- `mixed` → 236 / 347 ≈ 68%
- `soft_call` → 230 / 324 ≈ 71%
- `late_stage` → 125 / 169 ≈ 74%
- `sms_first` → 179 / 274 ≈ 65%

Вывод:
- `sms_first` хуже остальных по контактируемости
- остальные стратегии находятся примерно на одном уровне

### Конверсия в оплату
Доля оплаченных кейсов:
- `mixed` → 73 / 347 ≈ 21%
- `soft_call` → 65 / 324 ≈ 20%
- `late_stage` → 39 / 169 ≈ 23%
- `sms_first` → 50 / 274 ≈ 18%

Вывод:
- `late_stage` лучше конвертит кейсы в оплату
- `sms_first` снова показывает худший результат

### Средний платёж
- `late_stage` → ~14.9k
- `mixed` → ~14.6k
- `soft_call` → ~13.0k
- `sms_first` → ~11.8k

Вывод:
- `sms_first` даёт самый слабый средний чек
- `late_stage` и `mixed` выглядят сильнее по качеству сборов

### Collection rate (ключевая метрика)
- `late_stage` → 5.33%
- `mixed` → 5.03%
- `soft_call` → 4.51%
- `sms_first` → 3.57%

## Итог
### Лучшие стратегии
- `late_stage` — лучший общий результат: сильнее по `collection_rate_7d`, конверсии в оплату и среднему платежу
- `mixed` — стабильный сбалансированный результат

### Средняя стратегия
- `soft_call` — хорошая контактируемость, но слабее монетизация

### Худшая стратегия
- `sms_first` — ниже контактируемость, конверсия в оплату, средний платёж и итоговый `collection_rate_7d`

## Вывод
Стратегии `late_stage` и `mixed` выглядят наиболее эффективными в рамках 7-дневного окна.  
Стратегия `sms_first` требует пересмотра или дополнительного анализа причин слабой эффективности.
